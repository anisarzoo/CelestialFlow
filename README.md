# CelestialFlow 🌌

CelestialFlow is a high-performance, interactive 3D particle system built with Three.js. It features gesture recognition via MediaPipe, allowing users to manipulate particles with hand movements.

![CelestialFlow Preview](https://github.com/anisarzoo/CelestialFlow/raw/main/preview.png) *(Placeholder for preview image)*

## ✨ Features

- **High-Performance Rendering**: Optimized for HD/4K displays with advanced post-processing.
- **Gesture Control**: Use your hands to interact with the particle system (MediaPipe Hands).
- **Dynamic Shapes**: Toggle between various shapes like Sphere, Heart, Galaxy, DNA, and more.
- **Audio Reactive**: Particles react to ambient sound and music.
- **Customization**: Adjust speed, particle count, size, bloom intensity, and colors.
- **Presets**: Save and load your favorite configurations.
- **Clean Mode**: Hide UI elements for a pure visual experience.

## 🎮 Controls

### 🖐 Gesture Controls
- **✊ Fist**: Contract particles.
- **🖐 Open Palm**: Expand particles.
- **☝ 1 Finger**: Next shape.
- **✌ 2 Fingers**: Previous shape.
- **🤟 3 Fingers**: Toggle trails.
- **🤙 Pinch**: Particle burst!
- **✌ Hold 2s**: Take a screenshot.

### 🖱 Mouse/Touch
- **Double Tap/Click**: Toggle UI visibility.
- **Quality Toggle**: Switch between HD and Performance modes.
- **Presets Menu**: Access saved configurations.

## 🚀 Getting Started

Simply open `index.html` in a modern web browser.

### Requirements
- WebGL 2.0 compatible browser (Chrome, Firefox, Safari, Edge).
- Webcam (for gesture recognition).

- [MediaPipe](https://google.github.io/mediapipe/) - Gesture Recognition
